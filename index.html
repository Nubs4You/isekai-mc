Getting started

How to join

Step 01

Launch Minecraft

Open Minecraft Java Edition and go to Multiplayer from the main menu.

Step 02

Add the server

Click Add Server and enter isekai-mc.net as the server address.

Step 03

Connect & play

Select Isekai Network from your list and click Join Server. See you in-world!
